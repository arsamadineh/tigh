## [0.0.1-beta] - 2026-07-06

### اضافه شده
- انتشار اولیه موتور API تیغ
- سیستم مسیریابی تریه‌ای با پارامترهای مسیر و وایلدکارد
- کش LRU با پشتیبانی TTL و بی‌اعتبارسازی الگو
- محدودیت نرخ با سه استراتژی: Token Bucket، Sliding Window، Fixed Window
- قطع مدار خودکار با تشخیص و بازیابی خرابی
- سیستم middleware سلسله‌ای با CORS و timing دارالتحریر
- جمع‌آوری معیارهای جامع با صدک‌های تأخیر
- انطباق‌گر Next.js برای ادغام یکپارچه
- پشتیبانی کامل TypeScript با بررسی نوع سخت‌گیرانه
- مستندات گسترده فارسی و انگلیسی

### ویژگی‌های اصلی
- مسیریابی: مسیریابی تریه‌ای سریع با پشتیبانی از پارامترهای نام‌دار و وایلدکارد
- کش‌کردن: کش LRU با TTL و حدود حافظه قابل پیکربندی
- محدودیت نرخ: استراتژی‌های متعدد برای موارد مختلف
- قطع مدار: تشخیص خرابی خودکار با حالت بازیابی نیم‌باز
- Middleware: پشتیبانی middleware سفارشی با ابزارهای دارالتحریر
- معیارها: پیگیری درخواست، صدک‌های تأخیر، آمار کش، آمار محدودیت نرخ
- آماده برای Next.js: انطباق‌گر یکپارچه برای مسیرهای API
- عملکرد: بهینه‌شده برای سرویس‌های API با توان بالا

### مستندات
- README کامل در انگلیسی و فارسی
- راهنمای استفاده تفصیلی در انگلیسی و فارسی
- راهنمایی‌های سهم‌گذاری
- مستندات مرجع API
- مثال‌های کاربردی

### محدودیت‌های شناخته‌شده
- تنها برای یک فرآیند (پشتیبانی نمی‌کند از محدودیت نرخ توزیع‌شده یا کش)
- بدون ادغام پایگاه‌داده دارالتحریر
- معیارها تنها در حافظه ذخیره می‌شوند
